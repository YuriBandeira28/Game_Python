# Sobre:
Trabalho Realizado na disciplina de computação gráfica como atividade avaliativa

# Como rodar?

## Clone o projeto:
```bash
`git clone https://github.com/YuriBandeira28/Game_Python.git`
```

## Instale as dependências
```bash
pip install -r requeriments.txt
```

## Inicialize o Game

```bash
python main.py
```
Ou
```bash
python3 main.py
```
